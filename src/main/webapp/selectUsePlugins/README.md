##想到找这个demo的原因是
为解决select的size在手机浏览器不支持而生。
比如http://www.w3school.com.cn/tiy/t.asp?f=html_select_size

虽然这个项目对我没有用，因为不满足需求太重的项目。不过对于喜欢用插件的朋友还是挺有用的
项目效果图是这样的：http://www.17sucai.com/pins/demoshow/6518
